# Data Analytics Project

## Overview
This project demonstrates an end-to-end data analytics workflow, starting from raw data to actionable business insights. The project includes data loading, exploratory data analysis (EDA), data cleaning, SQL-based analysis, interactive dashboard creation in Power BI, and reporting through a presentation.

The objective is to transform raw data into meaningful insights that support data-driven decision-making.

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
```

### 2. Navigate to the project folder

```bash
cd your-repository
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Jupyter Notebook

```bash
jupyter notebook
```

Open the notebook in the `notebooks/` folder and execute the cells.

### 5. Execute SQL Queries

Run the SQL scripts using your preferred database:

- PostgreSQL
- MySQL
- SQL Server

### 6. Open the Dashboard

Open the `.pbix` file in **Power BI Desktop** to explore the interactive dashboard.
